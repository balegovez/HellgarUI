# HellgarUI

## Descripción
HellgarUI es un framework CSS personalizado, robusto y responsivo. Diseñado bajo el paradigma Mobile First y desarrollado íntegramente con SASS, este framework busca proporcionar una arquitectura limpia y componentes sofisticados para aplicaciones web con un enfoque profesional y corporativo.

## Paleta de Colores Oficial
HellgarUI utiliza una paleta vibrante y nostálgica inspirada en el universo Pokémon, balanceando colores energéticos con tonos neutros para mantener la limpieza visual:

* **Primario:** Azul Pokémon `#3B4CCA` - Para navbars y acciones principales.
* **Secundario:** Amarillo Pikachu `#FFDE00` - Para acentos y hover states.
* **Estados:** Verde Bulbasaur (`#4DAD5B`), Rojo Pokébola (`#CC0000`), Naranja Charmander (`#EE8130`) y Celeste Articuno (`#98D8D8`) para alertas y mensajes del sistema.
* **Neutros:** Tonos oscuros (`#1D2A42`) y claros (`#F8F9FA`) para garantizar la máxima legibilidad en textos y fondos.
